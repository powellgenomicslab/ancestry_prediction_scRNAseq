
Execute Steps Manually
=============================

We have provided the commands to run each single cell capture manually if your data don't match the assumptions of the snakemake pipeline, you want to alter some parameters of you just want to understand the steps that are being executed at each step.

Importantly, all the softwares and files required to run these steps manually are still provided in the Singularity image so no additional instllation will be necessary.

To run your dataset(s) manually, procded to the :doc:`ManualCommands` documentation.
